Actividad 2

Utilice Javascript para realizar las siguientes validaciones. En caso de que no pase la validación debe marcar el circulo de rojo (agregar clase rojo) en caso de que pase la validacion debe marcar el circulo de verde (agregar clase verde).

Una vez que pase todas las validaciones se debe enviar el formulario a la direccion ingresada en el mismo (destino).

  • El nombre no puede estar vacío, Debe tener al menos 2 caracteres. Debe comenzar con una letra mayúscula. No puede contener números ni caracteres especiales
  • El apellido puede estar vacío. Debe comenzar con una letra mayúscula. No puede contener números ni caracteres especiales
  • Si se ingresa uno de los numeros, se debe ingresar el segundo, no puede haber solo uno. Si se ingresa solo uno, debe mostrar un error.
  • El numero 1 debe ser menor que el numero 2 (si se ingresaron ambos)
  • La contraseña debe ser de entre 8 y 24 caracteres. Debe incluir al menos una letra mayúscula, una letra minúscula, un número y un carácter especial.
  • Ambas contraseñas coinciden.
  • Hay al menos 3 opciones seleccionadas.
